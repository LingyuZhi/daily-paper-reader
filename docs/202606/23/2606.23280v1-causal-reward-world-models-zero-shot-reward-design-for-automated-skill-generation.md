## 摘要
自动奖励设计（ARD）旨在通过语言驱动的奖励函数合成来替代强化学习中的手动奖励工程。然而，现有的基于大型语言模型（LLMs）的方法本质上仍由相关性驱动，依赖迭代的环境反馈来为每个具体任务优化奖励假设。这种范式不仅导致推理效率低下，而且使LLMs容易受到语义上看似合理但因果上虚假的奖励组件的影响，从而导致优化无效。为了解决这些局限性，我们提出了因果奖励世界模型（CRWM），该模型通过在多任务交互数据上进行离线预训练，显式地对候选奖励组件与任务目标物理变量之间的因果拓扑关系进行建模。基于从粗到细的预训练策略，我们引入了一个联合优化模块，该模块将显式机制解耦与置信度感知的软融合相结合，利用微观轨迹来优化粗结构先验，从而构建一个鲁棒且可解释的因果骨架。在推理过程中，LLMs利用CRWM作为任务无关的因果先验来约束奖励生成，实现零样本奖励函数设计。我们的工作为ARD问题开辟了一种新的白盒范式。在复杂连续控制基准上的大量实验表明，CRWM无需反馈驱动的奖励优化即可生成可执行的奖励函数，显著降低了获取新机器人技能的设计延迟，同时匹配或超越了最先进的性能，并进一步在未见过的任务和不同机器人形态上表现出强大的泛化能力。

## Abstract
Automated Reward Design (ARD) aims to replace manual reward engineering in reinforcement learning with language-driven reward function synthesis. However, existing approaches based on large language models (LLMs) remain inherently correlation-driven, relying on iterative environmental feedback to refine reward hypotheses for each specific task. This paradigm not only results in inefficient reasoning but also makes LLMs susceptible to semantically plausible yet causally spurious reward components, leading to ineffective optimization. To address these limitations, we propose the Causal Reward World Model (CRWM), which explicitly models the causal topological relationships between candidate reward components and task-targeted physical variables through offline pre-training on multi-task interaction data. Based on a coarse-to-fine pre-training strategy, we introduce a joint optimization module that integrates Explicit Mechanism Decoupling with Confidence-Aware Soft Fusion to refine coarse structural priors using micro-level trajectories, thereby constructing a robust and interpretable causal skeleton. During inference, LLMs leverage CRWM as a task-irrelevant causal prior to constrain the reward generation, enabling zero-shot reward function design. Our work opens up a new white-box paradigm for the ARD problem. Extensive experiments on complex continuous control benchmarks demonstrate that CRWM generates executable reward functions without feedback-driven reward refinement, significantly reducing the design latency for acquiring new robotic skills while matching or surpassing state-of-the-art performance, and further exhibits strong generalization capabilities across unseen tasks and diverse robotic embodiments.

- **核心问题**：现有基于大型语言模型（LLM）的自动奖励设计（ARD）方法本质上是相关性驱动的，依赖迭代的环境反馈来优化每个具体任务的奖励函数。这导致推理效率低下，且LLM容易引入语义上看似合理但因果上虚假的奖励组件，引发两种典型故障：**优化崩溃**（奖励信号无法提供有用梯度）和**规范博弈**（智能体利用虚假相关获得高奖励但未完成真正任务）。
- **研究动机**：在同一动力学系统中，不同任务的奖励组件往往共享因果结构。如果能从多任务交互数据中提取出可复用的**因果先验**，就能使LLM在无环境反馈的情况下零样本生成有效奖励，避免昂贵的试错循环。
- **整体意义**：提出因果奖励世界模型（CRWM），将ARD从一个依赖迭代精炼的“黑盒”相关范式，转变为基于显式因果结构的“白盒”预训练+零样本推理范式，显著降低新技能获取延迟，并提升泛化能力。

### 2. 方法论：核心思想、关键技术细节、算法流程
- **核心思想**：通过离线多任务预训练构建一个任务无关的因果骨架（CRWM），该骨架显式建模候选奖励组件（物理变量）与任务目标之间的因果拓扑关系。推理时LLM利用CRWM作为因果先验，通过“因果剪枝”零样本生成可执行奖励函数，无需任何环境反馈。
- **关键技术细节**：
  1. **层次化数据收集**：对预训练任务使用已有ARD框架抽取k个奖励组件，将其归一化为d个共享结构变量；随机采样N组权重向量（相当于介入操作）训练策略，同时收集**宏观数据集**（权重-最终完成任务分数）和**微观数据集**（每步结构变量轨迹）。
  2. **结构先验提取**：利用预训练因果基础模型（LimiX）从宏观数据估计因果效应矩阵，得到粗粒度的结构先验⟨M_prior, M_conf⟩（边缘存在性与置信度）。
  3. **联合优化模块（精炼）**：采用“从粗到细”策略，通过两个子模块优化：
     - **显式机制解耦（EMD）**：构建状态依赖的瞬时拓扑 `M_inst(s_{t-1}) = M_inv ⊙ (1 + M_trans(s_{t-1}; θ))`，其中M_inv为可学习的因果骨架（最终作为CRWM），M_trans为MLP参数化的瞬态物理交互分量。通过乘法门控，M_inv保持稳定，M_trans建模短时变化。
     - **置信度软融合（Soft Fusion）**：损失项 `L_soft = γ ||M_conf ⊙ (M_inv - M_prior)||_F^2`，对高置信度先验施加强正则，低置信度允许灵活偏离，避免继承结构幻觉。
  4. **优化目标**：联合最小化 `L_MSE + λh(M_inv) + L_soft`，其中h为DAGMA有向无环性正则。使用增广拉格朗日方法求解。
  5. **因果-ARD推理**：将收敛的M*_inv通过2-means聚类抽取活跃因果边，序列化为拓扑三元组（source, target, causal effect）构成Causal Topology Prompt，引导LLM生成奖励代码时对无因果路径到达score的组件系数置零（因果剪枝），在单轮推理中输出可执行奖励函数。

- **实验亮点**：
  - 同时提供模拟和真实实验，验证由模拟到真实的有效性。
  - 广泛的消融与敏感性分析，验证每个设计选择。
  - 多LLM和因果基础模型替换实验，证明方法稳定性。

### 8. 不足与局限
- **预训练成本高**：需要大量离线交互数据（4200次策略训练运行）和算力（8块A800），对于资源受限团队可能不友好。
- **组件池边界限制**：CRWM的泛化仅覆盖全局原子奖励组件池中的变量，若目标任务需要全新的物理原语（如流体动力学变量），需先扩展组件池并补充训练数据，无法直接零样本。
- **未进行双臂真实验证**：虽然模拟实验侧重双臂操作，但真实实验仅使用单臂移动机械臂（CASBOT W1），缺少真实场景下双臂协调任务的验证。
- **安全性保证未讨论**：论文未涉及奖励函数可能带来的安全风险或奖励黑客的检测机制。
- **随机性细节欠缺**：未明确报告每次实验的随机种子范围或置信区间，仅展示均值。
