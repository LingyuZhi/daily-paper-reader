## 摘要
在语言模型的强化学习（RL）中，估计token级别的优势仍然具有挑战性，因为扩大情节经验收集成本高昂。对于基线优势估计方法，难度进一步加大，因为重复采样会导致轨迹发散成本质不同的推理前缀。在这种情况下，像GRPO这样的RL算法被证明是有限的：结果奖励过于稀疏，无法归因于诸如中间步骤之类的具体动作，并且跨采样轨迹的比较并非易事，因为它们具有异质性。为了减轻重复采样的计算成本和信用分配的难度，我们研究了针对语言模型RL中token级别信用分配的单次采样近端策略优化（SR-PPO）。我们不是通过归一化候选组内的情节回报来估计优势，而是使用每个提示的一次采样中的蒙特卡洛结果来训练一个校准的token级别信用评论者。具体来说，我们使用评论者来预测提示前缀的Pass@k成功概率，该概率来自Pass@1尝试。这个选择产生的学习信号比Pass@1更具选择性：它降低了容易解决的提示前缀的权重，同时优先考虑那些成功概率仍然很小的困难前缀。我们证明，随着k的增加，Pass@k收敛到一个可达性指标，反映了一个前缀是否可以导致至少一个成功的延续。在显式状态图中，极限（k→∞）可以在O(|V|+|E|)时间内计算，为直接信用分配提供了一个有希望的替代方案，而无需采样对比轨迹。作为初步验证，SR-PPO表现出稳定的学习动态，并且在数学推理基准测试（如HMMT26和AIME24）上Pass@128成功率持续提升。

## Abstract
Estimating token-level advantages in reinforcement learning (RL) for language models remains challenging because scaling up episodic experience collection is expensive. The difficulty intensifies for baseline advantage estimation methods, where repeated sampling causes trajectories to diverge into substantially different reasoning prefixes. In this context, RL algorithms such as GRPO prove limited: an outcome reward is too sparse to be attributed to specific actions like intermediate steps, and comparisons across sampled traces are non-trivial because they are heterogeneous. To mitigate both the computational cost of repeated sampling and the difficulty of credit assignment, we study single-rollout proximal policy optimization (SR-PPO) featuring token-level credit assignment in RL for language models. Instead of estimating advantages by normalizing episodic returns within the candidate group, we train a calibrated token-level credit critic using Monte Carlo outcomes from one rollout per prompt. Specifically, we use the critic to predict the Pass@k success probability at the prompt prefix, which is derived from a Pass@1 attempt. This choice yields a more selective learning signal than Pass@1: it discounts easily solved prefixes while prioritizing hard ones whose success probability remains marginal. We show that as $k$ increases, Pass@k converges to a reachability indicator, reflecting whether a prefix can lead to at least one successful continuation. In an explicit state graph, the limit ($k \rightarrow \infty$) can be computed in $O(|V|+|E|)$ time, offering a promising surrogate for direct credit assignment without the need to sample contrastive traces. As an initial validation, SR-PPO exhibits stable learning dynamics, along with consistent gains in Pass@128 success rates on mathematical reasoning benchmarks such as HMMT26 and AIME24.

- **问题**：语言模型（LM）的强化学习（RL）中，常使用结果奖励（正确/错误）。但一个推理轨迹包含众多中间步骤，最终结果过于稀疏，难以进行 token 级别的信用分配（credit assignment）。特别是当需要多次采样（rollout）时，不同采样得到的推理前缀（prefix）高度异质性，导致基于组归一化的方法（如 GRPO）统计噪声大、计算成本高。
- **背景**：在数学推理、代码生成等任务中，正确结果可自动验证，但最终奖励无法区分哪些中间步骤有益或有害。现有工作如 GRPO、GAE-PPO 等在多采样场景下有效，但面对单次采样（single rollout）时性能下降。
- **整体含义**：作者致力于在**仅使用每个 prompt 的一次采样**这一严格约束下，实现有效的 token 级信用分配。这能显著降低采样成本，并适用于长程 agent 系统（如工具调用、多步推理）。

## 2. 论文提出的方法论

### 核心思想
- 训练一个**前缀级信用评论者（prefix credit critic）**，在单次采样的轨迹上，对每个前缀（prefix）预测最终成功的概率（Pass@k 值）。然后利用**局部值变化**（difference-in-value）作为 token 级的优势信号，并加上终端校正项（terminal correction），从而将稀疏的最终结果转化为稠密的 token 级信用。

### 关键技术细节
- **Pass@k 值族**：定义 qπ,k(st) = 1 − (1 − qπ(st))^k，其中 qπ(st) 是 Pass@1 成功概率。Pass@k 衡量从前缀出发，至少一次成功采样的概率。当 k>1 时，对已经容易成功的前缀梯度变小，对困难但可达的前缀保持敏感，从而产生更选择性、更稳定的信用信号。
- **信用评论者训练**：对每个前缀使用 Binary Cross-Entropy + Brier Score 损失，监督来自单次 rollout 的最终结果 Y。通过逆变换将 Pass@k 预测映射回 Pass@1 估计。
- **Token 级优势**：Aπ,k(st) = (ˆqπ,k(st) − ˆqπ,k(st−1)) + (Y − ˆqπ,k(sT))。第一项测量该 token 造成的预测成功概率变化，第二项确保总信号与真实结果一致。
- **策略优化**：采用单步策略梯度，等价于 on-policy 更新（无需裁剪）；每个 batch 使用 B 个 prompt 各一次 rollout，M 为总 token 数。
- **与可达性的联系**：证明当 k→∞ 时，Pass@k 趋近于可达性指标（是否至少有一条成功路径）。在已知状态图下，可达性可在 O(|V|+|E|) 时间内通过反向图遍历精确计算。

### 算法流程（文字描述）
1. 基于当前策略 πθ_old 对每个 prompt 采样一条完整轨迹。
2. 对轨迹中每个前缀，用信用评论者网络预测 Pass@k 值（如 Pass@4）。
3. 计算每个 token 的局部值差，加上终端校正，得到优势 Aπ,k(st)。
4. 计算重要性权重 ratio = πθ/πθ_old，最小化 -log πθ * A（单步更新，无裁剪）。
5. 用最终结果 Y 更新评论者网络（BCE + Brier 损失），使预测校准。
