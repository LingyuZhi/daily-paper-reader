## 摘要
通过强化学习训练的大语言模型智能体通常学习到脆弱、特定于任务的捷径。我们假设，当智能体的成功轨迹在结构上可压缩、分解为一组可重用的抽象模式时，其泛化能力更强。为形式化这一概念，我们引入ReuseRL，将智能体强化学习建立在最小描述长度（MDL）原则之上。ReuseRL从成功轨迹中提取共享技能词典，并通过分割代价增强强化学习目标，显式惩罚编码效果差的特异行为。我们证明了该压缩惩罚的PAC-Bayes泛化界。在ALFWorld、TextWorld-Cooking和Countdown-Stepwise上，ReuseRL相比原始GRPO和强回合长度基线，提升了分布内和分布外的成功率。

## Abstract
Large language model agents trained with reinforcement learning (RL) often learn brittle, task-specific shortcuts. We hypothesize that agents generalize better when their successful trajectories are structurally compressible, decomposed into a small set of reusable abstract patterns. To formalize this, we introduce ReuseRL, which grounds agentic RL in the Minimum Description Length (MDL) principle. ReuseRL extracts a shared skill dictionary from successful trajectories and augments the RL objective with a segmentation cost, explicitly penalizing idiosyncratic behaviors that encode poorly. We prove a PAC-Bayes generalization bound for this compression penalty. Across ALFWorld, TextWorld-Cooking, and Countdown-Stepwise, ReuseRL improves in- and out-of-distribution success over vanilla GRPO and strong round-length baselines.

## 1. 核心问题与整体含义

- **研究动机**：大语言模型（LLM）智能体通过强化学习（RL）训练时，常常学到脆弱、任务特定的捷径（即“推理崩溃”），导致泛化能力差。作者认为问题部分原因在于只优化任务成功信号，允许产生特异性的单次方案，这些方案无法被复用为可迁移的技能。
- **整体含义**：提出“结构可压缩性”假说——成功轨迹若能分解为一小组可重用的抽象模式（即结构可压缩），则泛化更强。基于最小描述长度（MDL）原则，将智能体RL与数据压缩联系起来，通过显式惩罚难以压缩的行为来促进技能复用。

## 2. 方法论

- **核心思想**：将成功轨迹投影为原子技能序列，从中提取共享技能字典，并利用字典编码代价作为RL中的正则项，激励策略使用可复用的技能模式而非异质序列。
- **关键技术细节**：
  - **技能投影**：将轨迹映射为原子技能序列（如“探索—拿取—运输—变换—交付”），通过规则化的动词映射实现。
  - **技能字典**：包含可重用子序列（短语），字典需包含所有原子技能以保证可编码性。
  - **分割代价（Segmentation Cost）**：给定字典，一条轨迹的分割代价 = 所需字典短语数 / 最大步数T。可以度量轨迹压缩程度。
  - **MDL目标**：批量成功轨迹的两部分描述长度 = 字典存储代价（平摊） + 逐个轨迹分割编码代价。通过EM风格交替优化：E步用BPE贪心近似最小化MDL提取字典；M步将分割代价作为惩罚项加入GRPO奖励信号。
  - **理论保证**：证明了PAC-Bayes泛化界，表明优化经验MDL目标能控制未来成功轨迹的期望描述长度。同时证明纯回合长度惩罚是固定码本的退化情况（命题1），而ReuseRL可区分相同长度但不同结构可压缩性的轨迹。
- **算法流程（文字描述）**：
  1. 对当前策略采样一批轨迹；
  2. 筛选成功轨迹，投影为技能序列；
  3. （可选）加入全局成功缓冲（FIFO）；
  4. 用BPE-最小描述长度（MDL）方法提取最优字典（E步）；
  5. 计算每个成功轨迹的分割代价，构造奖励：`R(τ) - λ·1{成功}·SegCost`；
  6. 用GRPO更新策略（M步）；重复。

## 6. 主要结论与发现

- 结构压缩（分割代价）普遍优于纯回合长度惩罚和原始GRPO，在三个基准的所有指标上取得最佳或次佳。
- 纯回合长度惩罚在TextWorld-Cooking上甚至低于GRPO，因为它过度惩罚必需的探索步骤（如寻找目标），陷入烧毁失败模式。而分割代价能保留可复用的结构（如探索→拿取→运输→交付），从而避免此类陷阱。
- 全局成功缓冲在多数场景下提升稳定性，但可能导致简单任务的主导模式抑制困难任务所需的结构（如TextWorld-Cooking困难拆分），去除缓冲反而更好。
- BPE贪心近似足够接近最优MDL，且运行效率高，适合在线使用。
- 技能复用与压缩正则化能有效提升智能体泛化性，验证了学习可复用结构优于盲目最小化步数。

## 7. 优点

- **理论扎实**：为压缩惩罚提供了PAC-Bayes泛化界，并与MDL原则建立严格联系。
- **方法简洁有效**：仅需在GRPO奖励中添加一项分割代价，无需额外教师蒸馏、外部记忆等复杂模块，可即插即用。
- **充分分析的消融**：通过合成实验验证BPE近似质量，通过有无全局缓冲对比分析了字典提取的稳定性与偏差。
- **案例深入**：对每个环境剖析失败模式（ALFWorld搜索不足、TW-Cooking烧毁、Countdown-Stepwise操作多样性），清晰说明分割代价如何避免纯长度惩罚的缺陷。
- **可解释性**：提取的技能短语与已知优秀技能库（如SkillRL）对应，表明学到的结构具有实际意义。

## 8. 不足与局限

- **依赖人工规则**：技能投影 `φ` 需要按环境手动定义动词映射（每个环境5~26个原子技能），迁移到新领域需额外设计和验证。
- **BPE近似局限**：贪心搜索在更大词汇表（|Σ| 增大）时可能距离最优解更远，需要更复杂的搜索策略。
- **假设条件限制**：定理依赖于“成功轨迹可共享可重用分解”的假设（Assumption 2），在无结构或纯新推理任务中可能不成立，此时分割代价退化到回合长度惩罚。
- **实验规模有限**：仅测试了1.5B~1.7B参数模型，未扩展到更大模型或视觉-语言智能体；训练仅300步，未探索长时间尺度的行为变化。
- **未与最新方法比较**：虽在引言和隐式MDL分析中提及SkillRL、ERL，但实验中没有直接与这些需要额外监督的强基线对比（论文说明是刻意隔离以验证压缩本身的作用），因此相对于最先进经验增强方法的绝对优势未知。
