## 摘要
现有的记忆增强型大语言模型智能体仅以文本摘要或检索段落的形式将过往经验存储在提示空间中，而在整个运行过程中保持模型参数不变。这类智能体能够“查阅”已见内容，但无法“从中学习”：其策略不因经验而改变，任何从上下文中丢弃的信息都将永久丢失。我们提出一种自进化参数记忆框架TMEM，其中智能体不仅将历史压缩为显式记忆，还通过轻量级在线更新将提炼后的监督信息吸收到快速LoRA权重Δ_t中，从而在单次任务中真正改变其未来行为。我们将此形式化为具有快速权重运行动态的智能体决策过程：动作从π_{θ_0+Δ_t}中采样，而提取动作产生用于更新Δ_t的监督信息以影响后续决策。这一视角使得提取策略可直接通过强化学习进行优化：训练θ_0不仅改进行动动作，还提升用于在线LoRA适应的数据质量。我们进一步提出基于SVD的LoRA子空间初始化以加速在线收敛。在LoCoMo、LongMemEval-S、多目标搜索和CL-Bench上的实验表明，TMEM在不同模型规模下均一致优于基于摘要和基于检索的基线方法。

## Abstract
Existing memory-augmented LLM agents store past experience exclusively in prompt space, as textual summaries or retrieved passages, while keeping model parameters frozen throughout a rollout. Such agents can \emph{look up} what they have seen but cannot \emph{learn from} it: their policy is unchanged by experience, and any information dropped from the context is permanently lost. We introduce \texttt{TMEM}, a self-evolving parametric memory framework in which the agent not only compresses history into explicit memory but also absorbs distilled supervision into fast LoRA weights $Δ_t$ via lightweight online updates, genuinely altering its future behavior within a single episode. We formalize this as an agentic decision process with fast-weight rollout dynamics: actions are sampled from $π_{θ_0+Δ_t}$, while extraction actions produce supervision that updates $Δ_t$ for subsequent decisions. This view makes the extraction policy directly optimizable by RL: training $θ_0$ improves not only task actions but also the quality of the data used for online LoRA adaptation. We further propose SVD-based initialization of the LoRA subspace to accelerate online convergence. Experiments on LoCoMo, LongMemEval-S, multi-objective search, and CL-Bench show that \texttt{TMEM} consistently outperforms summary-based and retrieval-based baselines across different model scales.

## 2. 方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- TMEM将智能体的记忆分为三部分：工作上下文 \(h_t\)、显式记忆 \(m_t\)（可选）、以及快速参数记忆 \(\Delta_t\)（由LoRA权重表示）。当上下文预算达到上限时，智能体进入记忆写入模式：将当前会话蒸馏为**基于QA的监督数据**，并通过轻量级在线SFT更新到 \(\Delta_t\) 中。后续动作由适应后策略 \(\pi_{\theta_0 + \Delta_t}\) 生成，从而在无需重复插入相同证据到提示中的情况下，让蒸馏后的经验通过快速权重影响未来推理。
- 方法将提取动作（memory-writing action）视为决策过程的一部分，通过强化学习（RL）直接优化基础模型参数 \(\theta_0\)，以产出更优质的监督数据用于后续在线LoRA适应。

### 关键技术细节
- **快速权重运行动态（Fast-Weight Rollout Dynamics）**：
  - 给定上下文预算 \(L_{\text{max}}\)，触发索引 \(t_i\) 处，智能体被追加一个记忆写入提示 \(d\)（图1），要求其生成结构化QA对。
  - 更新规则：\( (m_{t_i+1}, \Delta_{t_i+1}, h_{t_i+1}) = ( \emptyset, \mathcal{T}(\Delta_{t_i}, a_{t_i}), \emptyset )\)，其中 \(\mathcal{T}\) 是轻量在线SFT/LoRA更新。显式记忆被丢弃，内容全部通过参数记忆保持。
- **基于SVD的参数记忆初始化**：
  - 标准LoRA随机初始化子空间，在少量梯度步下效率低。TMEM对每个目标权重矩阵 \(W\) 做SVD：\(W = U\Sigma V^\top\)，取前 \(r\) 个右奇异向量构成投影矩阵 \(A_0 = \Sigma_r V_r^\top\)，并冻结 \(A_0\)，只训练系数矩阵 \(B\)（\(B_0 = 0\)，因此初始 \(\Delta_0 = 0\)）。
  - 理论证明（定理1）：SVD初始化的逼近误差上界不大于随机高斯初始化，严格更小当谱对齐系数 \(\rho_r > r/d_{\text{in}}\)。
- **策略优化（Stop-Gradient Policy Optimization）**：
  - 使用段式似然比分解，对RL目标 \(\nabla_{\theta_0} J(\theta_0)\) 进行估计（公式9），其中在线更新 \(\mathcal{T}\) 产生的 \(\Delta_t\) 被停止梯度（stop-gradient），从而梯度只回传到基础模型 \(\theta_0\) 的普通动作和提取动作上。

### 算法流程（文字说明）
1. 从任务提示 \(q\) 开始，维护 \(h_t, m_t, \Delta_t\)。
2. 在每个生成事件 \(t\)，如果当前上下文长度 \(\ell(h_t) + \ell(m_t) \leq L_{\text{max}}\)，执行普通交互：采样动作 \(a_t \sim \pi_{\theta_0+\Delta_t}(\cdot|q,h_t,m_t)\)，接收环境反馈 \(o_t\)，更新 \(h_{t+1} = (h_t, a_t, o_t)\)，\(\Delta_{t+1} = \Delta_t\)。
3. 当上下文超出 \(L_{\text{max}}\)，触发记忆写入：追加提示 \(d\)，采样提取动作 \(a_{t_i} \sim \pi_{\theta_0+\Delta_{t_i}}(\cdot|q, h_{t_i}, m_{t_i}, d)\)，解析为QA对，执行在线LoRA更新 \(\Delta_{t_i+1} = \mathcal{T}(\Delta_{t_i}, a_{t_i})\)，清除工作上下文和显式记忆。
4. 重复直到最终响应 \(a_T\)，接收奖励。
5. 在RL训练阶段，使用公式(9)更新基础参数 \(\theta_0\)，提升任务行为与记忆提取质量。
