## 摘要
基于可验证奖励的强化学习（RLVR）已成为提升LLM推理能力的标准方法。然而，现有的PPO风格信任区域机制仍然是位置无关的，对所有令牌独立施加均匀阈值。这种逐点处理方式在两个关键方面与自回归生成相冲突。首先，均匀阈值忽略了自回归的不对称性。早期阶段的偏差会产生复合的序列级漂移，导致静态阈值对早期分歧的约束不足，而对后期探索则过度限制。其次，孤立地评估令牌级分歧忽略了累积的前缀漂移，即无论条件历史已经从决策策略偏离多远，都赋予相同的分歧允许量。为了解决这一局限性，我们提出了CPPO（累积前缀散度策略优化），这是一种令牌级掩码规则，通过两种耦合机制使更新与有限时域策略改进界对齐。首先，位置加权阈值对早期位置施加更严格的限制，因为其影响持续更久，同时放宽对后期令牌的约束。其次，累积前缀预算跟踪历史偏差，动态限制进一步的令牌级偏差，以防止沿前缀产生复合误差。实验上，CPPO提升了训练稳定性，并在不同模型规模下显著提高了推理准确性。

## Abstract
Reinforcement learning with verifiable rewards (RLVR) has become standard for improving LLM reasoning. However, existing PPO-style trust-region mechanisms remain position-agnostic by enforcing uniform thresholds across all tokens independently. This pointwise treatment conflicts with autoregressive generation in two critical ways. First, uniform thresholds ignore autoregressive asymmetry. Early-stage deviations produce compounding sequence-level drift, causing static thresholds to under-regulate early divergence and excessively constrain late-stage exploration. Second, evaluating token-level divergence in isolation overlooks cumulative prefix drift, granting the same divergence allowance regardless of how far the conditioning history has already deviated from the rollout policy. To address this limitation, we propose CPPO (Cumulative Prefix-divergence Policy Optimization), a token-level masking rule that aligns updates with a finite-horizon policy-improvement bound via two coupled mechanisms. First, a position-weighted threshold imposes stricter limits at early positions whose effects persist longer, relaxing constraints for late-stage tokens. Second, a cumulative prefix budget tracks historical deviations, dynamically restricting further token-level deviation to prevent compounding errors along the prefix. Empirically, CPPO enhances training stability and significantly improves reasoning accuracy across various model scales.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：在LLM强化学习（RLVR）中，PPO/GRPO等算法通过信任区域机制控制策略更新幅度，但现有方法采用**位置无关的均匀令牌级阈值**。这种处理方式忽略了自回归生成的两个关键特性：
  - **自回归不对称性**：早期令牌的偏差会通过后续步骤级联放大，产生更大的序列级分布漂移；而均匀阈值对早期偏差约束不足，对后期探索过度限制。
  - **累积前缀漂移**：令牌级偏差会在前缀中逐步累积，当历史条件已显著偏离滚动策略时，当前令牌的微小偏差都可能引入复合风险；但均匀阈值不感知历史上下文，赋予每个令牌相同的允许偏差量。
- **整体含义**：要提升RLVR训练的稳定性和推理性能，必须设计**位置感知的、能够动态调整的信任区域机制**，使约束强度与令牌位置和历史漂移程度相匹配。

## 2. 论文提出的方法论

### 核心思想
提出**CPPO（累积前缀散度策略优化）**，本质是一个**令牌级二元掩码**，在PPO/GRPO的标准比率-优势目标基础上，通过修改掩码条件来实现两种耦合的信任区域约束：

- **位置加权令牌级阈值**：设定一个递减的线性位置权重 \(w_t\)（从第一个令牌的1递减到最后一个令牌的 \(w_{\min}\)），实际约束 \(w_t D_t \le \delta\)。由于 \(D_t\) 是令牌级散度，等价于允许的散度阈值 \(\delta / w_t\) 在早期更严格、后期更宽松，匹配了早期偏差影响更长的剩余序列这一事实。
- **累积前缀预算**：定义加权前缀散度和 \(S_t = \sum_{j=1}^t w_j D_j\)，要求 \(S_t \le \delta + \delta_b W_{t-1}\)（其中 \(W_{t-1}\) 是累积权重）。当历史平均散度超过阈值 \(\delta_b\) 时，当前令牌的有效阈值会进一步收紧，从而限制累积漂移。

### 关键技术细节
- **令牌级散度近似**：采用DPPO的Top-K减缩TV方法（K=20），在可接受计算成本下近似全词汇表的TV散度。
- **掩码规则**：令牌更新仅在满足以下条件时允许：
  - 有利地移动到策略（\(\hat{A}_t(\rho_t-1) \le 0\)），或者
  - 同时满足 \(w_t D_t \le \delta\) 和 \(S_t \le \delta + \delta_b W_{t-1}\)。
- **算法流程（单个响应）**：
  1. 初始化前缀和 \(S_0=0, W_0=0\)。
  2. 对每个位置 \(t=1..T\)：
     - 计算 \(w_t\)（线性衰减），\(Z_t = w_t D_t\)。
     - 计算有效阈值 \(c_t = \min(\delta,\, \delta + \delta_b W_{t-1} - S_{t-1})\)。
     - 若 \(\hat{A}_t(\rho_t-1) \le 0\) 或 \(Z_t \le c_t\)，则掩码 \(M_t=1\)（允许更新）；否则为0。
     - 更新前缀和：\(S_t = S_{t-1}+Z_t\)，\(W_t = W_{t-1}+w_t\)。

### 理论依据
论文推导了有限时域策略改进界（Theorem 1），表明通过前缀约束可将剩余残差从 \(O(T^2 \delta^2)\) 降低至 \(O(T^2 \bar{\ell} \delta_b)\)，当 \(\delta_b < \delta\) 时可获得更紧的界。
