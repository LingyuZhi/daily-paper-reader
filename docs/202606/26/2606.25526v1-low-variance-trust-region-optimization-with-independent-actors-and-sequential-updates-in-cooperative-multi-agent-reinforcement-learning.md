## 摘要
合作多智能体强化学习假设每个智能体共享相同的奖励函数，并且可以有效地使用单智能体的信任域框架进行训练。独立执行体设置不依赖其他智能体的动作，而是考虑每个智能体仅基于其局部信息行动，因此具有更灵活的应用。然而，在顺序更新框架中，需要在每个个体智能体的策略步骤后重新估计联合优势函数。尽管重要性采样在实践中取得了成功，但更新后的优势函数存在指数级的高方差问题，这可能导致不稳定的收敛。在这项工作中，我们首先从经验和理论上分析了高方差优势。为了克服这一限制，我们引入了一个裁剪目标来控制顺序更新中优势波动的上界。通过提出的目标，我们提供了一个具有亚线性收敛到ε-纳什均衡的单调界。我们进一步使用我们的裁剪目标推导出两种新的实用算法。在三个流行的多智能体强化学习基准上的实验结果表明，我们的方法在大多数环境中优于测试的基线。通过仔细分析不同的训练设置，我们的方法以稳定的收敛特性和所需的低优势方差估计而突出。为了可重复性，我们的源代码公开在https://github.com/giangbang/Low-Variance-Trust-Region-MARL。

## Abstract
Cooperative multi-agent reinforcement learning assumes each agent shares the same reward function and can be trained effectively using the Trust Region framework of single-agent. Instead of relying on other agents' actions, the independent actors setting considers each agent to act based only on its local information, thus having more flexible applications. However, in the sequential update framework, it is required to re-estimate the joint advantage function after each individual agent's policy step. Despite the practical success of importance sampling, the updated advantage function suffers from exponentially high variance problems, which likely result in unstable convergence. In this work, we first analyze the high variance advantage both empirically and theoretically. To overcome this limitation, we introduce a clipping objective to control the upper bounds of the advantage fluctuation in sequential updates. With the proposed objective, we provide a monotonic bound with sub-linear convergence to $ε$-Nash Equilibria. We further derive two new practical algorithms using our clipping objective. The experiment results on three popular multi-agent reinforcement learning benchmarks show that our proposed method outperforms the tested baselines in most environments. By carefully analyzing different training settings, our proposed method is highlighted with both stable convergence properties and the desired low advantage variance estimation. For reproducibility purposes, our source code is publicly available at https://github.com/giangbang/Low-Variance-Trust-Region-MARL.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **问题背景**：在合作多智能体强化学习中，独立智能体（independent actors）的“顺序更新”方案（sequential update）已被证明具有理论上的单调改进保证（如HATRPO/HAPPO）。但该方案在更新后期需要利用重要性采样重新估计联合优势函数，导致**优势估计方差呈指数级增长**，从而造成训练不稳定甚至发散。
- **研究动机**：作者从理论和实验两方面观察到，由于各智能体策略更新方向趋同，重要性采样比率的乘积会随智能体数量指数膨胀，使得后期智能体的优势估计方差极高。这一现象在MuJoCo等连续控制任务中已被实证（方差可达10^6）。现有方法（如MAPPO、HAPPO、CoPPO）未能从根源上解决此问题，且缺乏理论保证。
- **整体含义**：本文通过引入一个简单的**裁剪目标（clipping objective）** 来控制优势波动的上界，在降低方差的同时保留了单调改进性质和次线性收敛到ε-纳什均衡的保证。最终提出两个实用算法clip-HAPPO和clip-HATRPO，在多个基准上超越了现有方法。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：在顺序更新的每一步中，对重要性采样比率进行裁剪（clipping），避免其过度增长。具体地，将标准优势估计中的比率项ℜ（累积比率）限制为不大于1，从而保证方差有界。
- **关键技术细节**：
  - 定义新的代理损失函数（公式Definition 4）：
    \[
    L_{\pi}^{1:m,\text{clip}}(\tilde{\pi}^{1:m-1}, \hat{\pi}^m) = \mathbb{E}_{s\sim d_\rho^\pi, a\sim\pi}\left[ \left(\frac{\hat{\pi}^m(a^m|s)}{\pi^m(a^m|s)} - 1\right) \min\left(\mathfrak{R}_{\tilde{\pi},\pi}^{1:m-1}(s,a), 1\right) A^\pi(s,a) \right]
    \]
    其中 \(\mathfrak{R}^{1:m-1}\) 是先前智能体的累积重要性采样比率。裁剪操作将其限制在1以下。
  - 在信任区域优化中，KL散度采用逆序（reverse KL）并以期望形式惩罚，保证收敛分析成立。
- **算法流程**（Algorithm 1）：
  1. 初始化均匀随机策略 \(\pi^0\)。
  2. 对每一轮迭代 \(k=0,\dots,K-1\)：
     - 计算优势函数 \(A^{\pi_k}(s,a)\)。
     - 随机排列所有智能体。
     - 对每个智能体 \(m=1,\dots,n\) 依次更新策略：
       \[
       \pi^{m}_{k+1} \leftarrow \arg\max_{\pi^m} \left[ L_{\pi_k}^{1:m,\text{clip}}(\pi^{1:m-1}_{k+1}, \pi^m) - \beta_k \mathbb{E}_{\rho}^{\pi_k} D_{KL}(\pi^m \| \pi^m_k) \right]
       \]
  3. 输出均匀采样的策略 \(\bar{\pi}\)。
- **理论贡献**：
  - **Lemma 3（策略改进界）**：新目标函数仍能保证单调改进，只是多了一项与策略变化乘积相关的误差项。
  - **Theorem 4（收敛速率）**：算法以 \(O(\bar{\epsilon}\sqrt{N\log A}/(1-\gamma)\sqrt{K})\) 的速率次线性收敛到ε-纳什均衡。证明依赖于两步引理（one-step descent lemma和difference lemma），并假设策略始终具有正熵（状态可到达性）。
