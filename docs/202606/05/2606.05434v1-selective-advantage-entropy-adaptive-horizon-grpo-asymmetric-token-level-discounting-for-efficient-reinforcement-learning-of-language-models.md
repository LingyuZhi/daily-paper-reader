## 摘要
群体相对策略优化（GRPO）已成为一种有效的强化学习算法，用于在推理任务上对齐语言模型，但它对称地处理每个令牌位置和每个采样轨迹。我们引入了两种互补的扩展：（i）自适应地平线GRPO（AH-GRPO），它使用基于累积熵的折扣对每个令牌的策略梯度进行加权，在模型不确定时减少有效地平线；（ii）选择性优势AH-GRPO（SA-AH-GRPO），它仅对负优势轨迹应用此折扣，而保留正优势的成功轨迹不衰减。我们在GSM8K数学推理基准上评估了标准GRPO（alpha = 0）、AH-GRPO（alpha = 0.5）和SA-AH-GRPO（alpha = 0.5），使用了Qwen 2.5-1.5B-Instruct和Qwen 2.5-3B-Instruct模型，这些模型通过LoRA进行微调。在3B模型上，SA-AH-GRPO在第30步达到峰值Pass@1 = 0.858，并在第180步保持0.846，训练方差降低至0.0246，相对于GRPO降低了3.6倍，同时与其峰值准确率相当。在1.5B模型上，SA-AH-GRPO达到了0.686的峰值Pass@1，优于0.637的零样本基线。我们的分析表明，非对称折扣保留了正确解上的完整梯度信号，防止了熵坍塌，并显著稳定了训练，表明这是一种结构化生成任务上具有可验证奖励的强化学习的原理性归纳偏置。

## Abstract
Group Relative Policy Optimisation (GRPO) has emerged as an effective reinforcement-learning algorithm for aligning language models on reasoning tasks, but it treats every token position and every sampled rollout symmetrically. We introduce two complementary extensions: (i) Adaptive-Horizon GRPO (AH-GRPO), which weights each token's policy gradient using a cumulative entropy-based discount that reduces the effective horizon when the model is uncertain, and (ii) Selective-Advantage AH-GRPO (SA-AH-GRPO), which applies this discounting only to negative-advantage rollouts, leaving positive-advantage, successful trajectories unattenuated. We evaluate standard GRPO with alpha = 0, AH-GRPO with alpha = 0.5, and SA-AH-GRPO with alpha = 0.5 on the GSM8K mathematical reasoning benchmark using both Qwen 2.5-1.5B-Instruct and Qwen 2.5-3B-Instruct fine-tuned with LoRA. On the 3B model, SA-AH-GRPO achieves Pass@1 = 0.858 at its peak at step 30 and maintains 0.846 at 180 steps, with training variance reduced to 0.0246, a 3.6 times reduction relative to GRPO while matching its peak accuracy. On the 1.5B model, SA-AH-GRPO achieves a peak Pass@1 of 0.686, improving over the zero-shot baseline of 0.637. Our analysis shows that asymmetric discounting preserves the full gradient signal on correct solutions, prevents entropy collapse, and substantially stabilises training, suggesting a principled inductive bias for reinforcement learning with verifiable rewards on structured generation tasks.

# 论文中文总结

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：标准 GRPO（Group Relative Policy Optimisation）在用于语言模型推理任务的对齐时，对称地处理每一个 token 位置和每一个采样轨迹（rollout），没有区分模型在不同 token 上的不确定性，也没有对成功和失败的轨迹采用不同的梯度权重。这种对称处理导致训练不稳定，尤其是在高熵（不确定）的 token 位置上，噪声梯度可能破坏学习；同时成功轨迹中本应完整保留的梯度信号也可能被不必要地衰减。
- **研究动机**：受决策理论和课程学习直觉启发，作者认为（1）高熵 token 位置的梯度信号是噪声的，应被降低权重；（2）成功轨迹（正优势）的每个 token 都应获得完整梯度，而失败轨迹（负优势）中高熵位置的梯度才应被折扣。现有方法（如过程奖励模型、固定启发式加权）需要额外奖励模型或规则，而本文利用模型自身预测熵作为天然信号，无需额外监督。
- **整体含义**：本文提出一种**非对称、基于熵的自适应 token 级折扣**方法，用于强化学习中的语言模型微调，显著提升训练稳定性和最终准确率。

## 2. 论文提出的方法论：核心思想、关键技术细节

### 核心思想
通过在每个 token 位置引入一个取决于模型累积归一化熵的折扣权重，缩短在不确定区域上的有效梯度视界；并且**仅对负优势轨迹**施加该折扣，正优势轨迹保留全权重（权重恒为1）。

### 关键技术细节

- **归一化熵**：  
  对于 token t 的归一化熵定义为：
  \[
  \tilde{H}^{(i)}_t = \frac{H(\pi_\theta(\cdot | q, o_{i,<t}))}{\log V} \in [0,1]
  \]
  其中 H 是 Shannon 熵（以纳特为单位），V 是词汇表大小。实际上使用 top-K=500 的 logits 近似计算以节省内存。

- **自适应折扣权重**：  
  每个 token 的步折扣因子 \(\gamma^{(i)}_t = e^{-\alpha \tilde{H}^{(i)}_t}\)，累积权重为：
  \[
  w^{(i)}_t = \prod_{s=1}^t \gamma^{(i)}_s = \exp\left(-\alpha \sum_{s=1}^t \tilde{H}^{(i)}_s\right)
  \]
  通过在 log 空间中累积和实现数值稳定。当 α=0 时退化为标准 GRPO（权重恒为1）；α>0 时高熵前缀后折扣更强，缩短有效视界；α<0 时（熵放大）反效果。

- **AH-GRPO**：对所有 rollout 统一应用上述折扣权重，损失函数为加权版本，分母用有效加权 token 数归一化。

- **SA-AH-GRPO**：引入指示变量 \(n_i = 1[\hat{A}_i < 0]\)，选择性权重：
  \[
  \tilde{w}^{(i)}_t = n_i \cdot w^{(i)}_t + (1-n_i) \cdot 1
  \]
  即仅对负优势 rollout 使用熵折扣，正优势 rollout 的每个 token 权重恒为1。损失函数结构与 AH-GRPO 相同，但用该选择性权重。

- **奖励函数**：复合奖励，包含正确性（rcorrect，±0.5/0/1.5/4.0）、格式（rformat，-0.5~1.5）、标签存在性（rpresent，0/0.3/1.0）、步骤密度（rsteps，0~1.0），最大总奖励7.5。鼓励正确性及结构化输出的推理。

- **算法流程**（伪代码见 Appendix B）：对每个 prompt 采样 G 个完成；计算奖励和 group-normalized 优势；对每个 rollout 每个 token 计算归一化熵和累积权重；对正优势 rollout 设权重为1，负优势 rollout 保留熵折扣权重；计算加权裁剪代理损失 + KL 惩罚项；更新策略参数。

## 3. 实验设计：数据集、Benchmark、对比方法

- **数据集和 Benchmark**：GSM8K 数学推理基准（小学数学应用题）。使用 800 个训练样本（官方训练集的随机子集），500 个测试样本（官方测试集）。评估指标为 Pass@1（贪心解码，与标准答案完全匹配），报告 ±95% 置信区间。
- **模型**：Qwen 2.5-1.5B-Instruct（1.5B 参数）和 Qwen 2.5-3B-Instruct（3B 参数），使用 LoRA 进行参数高效微调（rank=16，alpha_LoRA=32，dropout=0.05），目标模块包括所有 query/key/value/output/MLP 投影层。
- **对比方法**：
  - 标准 GRPO（α=0，即无熵折扣）
  - AH-GRPO（α=0.5，对所有 rollout 应用折扣）
  - SA-AH-GRPO（α=0.5，仅负优势应用折扣）
  - 零样本基线（原始 instruct 检查点，保守调整后：1.5B 为 0.637，3B 为 0.831）
- **超参数**：学习率 5e-6，cosine 调度，weight decay 0.01，梯度裁剪 1.0，KL 惩罚 β=0.04，PPO clip ϵ=0.2，每步 batch 4 prompts，每 prompt 4 个完成，梯度累积 2 步，最大生成 token 512，熵 top-K=500，奖励阈值 0.15。
- **训练步数**：GRPO 和 AH-GRPO 训练 150 步，SA-AH-GRPO 训练 180 步以评估长期稳定性。每 15 步保存 checkpoint，每 30 步评估一次。
- **α 消融实验**：在 1.5B 模型上对 AH-GRPO 进行 α ∈ {-0.25, 0.0, 0.10, 0.25, 0.50} 的扫描（150 步），其中 α=0.0 为 GRPO 基线，负值表示熵放大。

## 6. 论文的主要结论与发现

- **准确率**：
  - 3B 模型上，三种 RL 方法都超越了零样本基线（0.831），峰值在 0.844-0.862 之间。SA-AH-GRPO 峰值 0.858，最终 0.846，与 GRPO 相当，但**训练方差显著降低**（0.0246 vs 0.0885，降低 3.6 倍）。

- **隐式课程学习**：随着训练进行，模型熵降低，折扣自然趋弱，早期保护优化器免受噪声影响，晚期允许更强更新。

## 7. 优点：方法或实验设计上的亮点

- **创新性**：首次将**基于模型自身不确定性的 token 级折扣**引入 RLVR，并**非对称地基于优势符号**应用，无需额外奖励模型或手工规则，是一种原理性的归纳偏置。
- **实用性**：直接利用现有 GRPO 框架，计算开销小（top-K 熵近似），易于实现。
- **实验设计稳健**：采用复合奖励函数鼓励正确性和结构化输出，避免模型仅匹配最终答案；使用 LoRA 参数高效微调，适合计算资源有限场景；报告统计置信区间，透明度好。
- **消融实验有深度**：不仅测试了正 α，还测试了负 α（熵放大），验证了方向性正确，并揭示了 α 大小与早熟/稳定性的折中。
- **结果显著**：在 3B 模型上 3.6 倍方差降低同时保持准确率，在 1.5B 模型上提升约 5 个百分点，表明方法在不同规模上均有效。
