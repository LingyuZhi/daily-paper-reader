## 摘要
强化学习已成为大语言模型后训练的核心范式。现有的无评论家强化学习方法通常为同一问题生成一组轨迹，以估计用于优势计算的价值基线。然而，这种设计存在数据效率低、分组同步障碍以及与结构化轨迹不灵活等缺陷。本研究重新审视了“分组”的作用，表明其核心功能并非仅仅是估计基线，而是防止对负样本的错误惩罚。基于这一见解，我们提出了负令牌过滤策略，这是一种简单有效的方法，能够实现稳定的单轨迹训练。我们将该方法应用于两种批次级优势方法，在与基于分组的强化学习技术相比时，在推理任务上取得了相当的性能，在智能体任务上取得了更强的性能。

## Abstract
Reinforcement learning (RL) has become a central paradigm for post-training large language models. Existing critic-free RL methods typically generate a group of rollouts for the same question to estimate value baselines for advantage computation. However, this design suffers from data inefficiency, group synchronization barriers, and inflexibility with structured rollouts. In this work, we revisit the role of the ``group'' and show that its underlying function is not merely to estimate baselines but to prevent false penalties on negative samples. Building on this insight, we propose negative token filtering, a simple and effective strategy that enables stable single-rollout training. We apply it to two batch-level advantage methods, achieving comparable performance on reasoning tasks and stronger performance on agentic tasks relative to group-based RL techniques.

- **核心问题**：现有的无评论家强化学习（critic-free RL）方法（如GRPO、RLOO、ReMax）在LLM后训练中，依赖为每个prompt生成多个rollout并组成group来计算优势（advantage）的基线。这种做法带来数据效率低、同步障碍、难以处理结构化rollout（如agentic任务）等问题。

- **整体含义**：重新定义group的角色，提出一种无需group的单rollout训练方法——负token过滤（Negative Token Filtering, NTF），从而在保持训练稳定性的同时提升效率。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

### 核心思想
- 单rollout训练的不稳定性主要源于负样本中高概率的“支持性token”被错误惩罚。通过过滤这些高概率token，可以消除有害更新，实现稳定训练。

### 关键技术细节
1. **负token过滤（NTF）**：对于每个负轨迹，按当前策略的token概率从高到低排序，mask掉前τ比例（默认τ=0.1）的最高概率token。剩余token的梯度按原序列长度归一化。公式：
   - 负项损失：\( l^-(o) = \frac{1}{|o|} \sum_{t \in K(o)} \max(\rho(o_t), 1-\epsilon) \)，其中K(o)为保留的token集（低概率部分）。
2. **应用方法**：将NTF应用于两种batch-level优势计算方法：
   - **REINFORCE++ (RF++)**：使用批次内z-score归一化奖励。
   - **Contrastive-REINFORCE (C-RF)**：去掉RF++中的权重系数，直接对正负样本的平均对数概率求差。
3. **训练流程**：每个prompt仅采样一个rollout，计算批次级优势，对负轨迹应用NTF后更新策略。
