## 摘要
主动式任务导向型代理必须自主预判用户需求，识别可执行的机会，并在恰当的时刻触发软件操作——这从根本上区别于等待明确指令的被动式系统。然而，现有方法缺乏可泛化的端到端方案来衡量和优化此类预判行为。本文提出ProActor，一个面向对话式任务调度的统一框架，它整合了：(1) 一种领域无关的自动标注方法，通过生成完整的机会时间窗口而非僵化的点标签，实现可扩展的主动性强化学习；(2) 系统化的主动性度量指标，同时捕捉时序质量和参考动作对齐度；(3) 利用GRPO及多种奖励设计的RL优化。我们的核心见解是：结合主动性评分细则的基于RULER的奖励对于提升时序质量至关重要，且由阶段感知的复合奖励所实现的主动性优化是平衡时序质量与参考动作对齐度的关键。时序感知RL需要广泛探索，因而要求高效的基础设施。我们开发了ART-F，一个将请求自适应推理集群与基于DDP的单节点多GPU训练相结合的自适应框架，使得4位量化的Qwen2.5-14B-ProActor-Q4模型能够以4-8倍的加速进行LoRA训练。在两个新自动标注数据集上的实验表明，主动时序得到显著改善，同时动作一致性可与当前最优基线相媲美。消融实验验证了不同复合奖励变体的有效性。

## Abstract
Proactive task-oriented agents must autonomously anticipate user needs, identify actionable opportunities, and trigger software actions at appropriate moments - fundamentally shifting from reactive systems that await explicit instructions. However, existing approaches lack generalizable end-to-end solutions for measuring and optimizing such anticipatory behaviors.   This paper introduces ProActor, a unified framework for conversational task scheduling that integrates: (1) a domain-agnostic automated annotation methodology that enables scalable proactiveness reinforcement learning (RL) by generating full opportunity time windows instead of rigid point labels, (2) systematic proactiveness metrics capturing both timing quality and reference action alignment, and (3) RL optimization using GRPO with various reward designs. Our insight is that RULER-based rewards with proactiveness rubrics are crucial for improving timing quality, and that proactiveness optimization enabled by stage-aware composite rewards is key to balancing timing quality and reference action alignment.   Timing-aware RL requires extensive exploration, demanding efficient infrastructure. We develop ART-F, an adaptive framework combining request-adaptive inference clusters with DDP-based training on single-node multi-GPU systems, enabling LoRA training of 4-bit Qwen2.5-14B-ProActor-Q4 with 4-8x speedups. Experiments on two newly auto-annotated datasets demonstrate significant improvements in proactive timing while maintaining action consistency comparable to state-of-the-art (SOTA) baselines. Ablations validate the effectiveness of distinct composite reward variations.

## 论文总结

### 1. 核心问题与整体含义
- **研究动机**：当前对话式AI代理大多是被动反应式系统，需等待用户明确指令后触发操作。向主动预判转型（主动代理）能在客户服务、企业自动化、实时辅助等场景中提前捕捉机会、无缝介入，从而提升交互效率与用户体验。
- **核心挑战**：
  - 缺乏领域无关、可扩展的自动标注方法，难以量化主动行为的“窗口期”而非单一正确时刻。
  - 现有评估指标未能系统捕捉时序质量与参考动作对齐度，导致无法有效优化主动时机。
  - 强化学习后训练（RL）需要大量探索，现有基础设施在量化模型上效率低下，难以大规模部署。
- **论文目标**：提出统一框架ProActor，实现对话级任务调度中“何时主动”的时序感知强化学习，并配套高效训练基础设施。

### 2. 方法论
ProActor由三大模块组成：
- **自动标注流水线**：
  - 将异构工具接口统一为标准化的动作目录（MCP工具、REST API等）。
  - 利用具有完整对话历史（含未来轮次）的“神谕”LLM标注器，在每个对话轮次生成参考动作及其就绪时间窗口（非僵化点标签），支持后续RL训练时的多时机探索。
- **主动性度量体系**：
  - **动作一致性**：AC（平均对齐）、Max AC（最佳单动作对齐）、差异度（衡量预测稳定性）。
  - **主动时序**：Proactive Timing（提前于参考窗口的比例）、Fault Trigger Rate（误触发率）、Ready Action Rate（动作就绪比例）。
  - 辅助指标包括信息一致性、动作依赖对齐等。
- **强化学习优化**：
  - 将任务调度形式化为对话轮次级别的GRPO优化问题，避免长对话的信用分配难题。
  - 奖励设计：
    - **基于指标的奖励**：RAC、Max RAC直接从参考动作对齐度计算。
    - **基于RULER的奖励**：通用RULER及自定义RULER（显式编码主动时机评价细则），由LLM评委对轨迹进行偏好评分。
    - **复合奖励**：加权组合指标与RULER奖励，并引入阶段感知自适应奖励（训练初期鼓励探索、中期平衡、后期保守），动态调整权重以平衡动作一致性与时序积极性。
- **高效训练框架（ART-F）**：
  - **自适应推理集群**：单GPU上启动多个vLLM服务器实例，通过负载均衡提升rollout吞吐。
  - **异步DDP训练**：主-从架构，支持对称/非对称数据划分与动态批次，最大化GPU利用率。
  - 支持MLflow/W&B实验跟踪、自定义RULER模块及Databricks兼容性。

### 3. 实验设计
- **数据集**：
  - ABCD+（已有动作执行日志的客服场景）：7,042段对话（5,647/703/692训练/验证/测试），平均每对话约21.2轮，共约11.5万条标注动作。
  - Home Loan（无动作日志的抵押咨询）：968段对话（774/97/97），平均约47.4轮，共约3万条标注动作。
- **Baselines**：
  - 非推理模式（直接预测）、推理模式（显式<think>再输出）、推理+动作状态图（ASG，跨轮次跟踪状态）。
  - 模型覆盖GPT-5.1、Gemini-2.5-flash、Claude-sonnet-4、Qwen2.5-14B-Instruct。
  - 还对比了SFT变体（LoRA微调与全参数微调）以验证RL的优势。
- **RL训练模型**：Qwen2.5-14B-ProActor-Q4（4-bit量化+LoRA）。
- **评价指标**：综合排名指数PRI（调和平均一致性指数与时机指数），以及各分解指标。
- **消融实验**：不同奖励类型对比（RAC、Max RAC、自定义RULER、加权/自适应复合奖励），λ_max 对自适应RULER的影响，不同LLM评委模型稳健性测试，数据量缩放效果等。

### 4. 资源与算力
- **RL训练**：
  - ABCD+：4×H200 GPU（每卡141 GB显存），训练约3.5~5.7天。
  - Home Loan：8×H100 GPU（每卡80 GB），训练约1.5~2.15天。
- **模型规格**：Qwen2.5-14B-Instruct 4-bit量化，LoRA rank=8，alpha=16，最大上下文长度9216 tokens。
- **推理加速**：ART-F框架通过多服务器并行、动态批次与并行持久化，实现了4~8倍训练加速。

### 5. 实验数量与充分性
- 共涉及**两大领域数据集**、**十余种基线配置**（含不同模型与提示策略）、**多种奖励设计**（至少6种）、**多组消融实验**（奖励类型、数据量、λ_max、评委模型等），并包含SFT直接对比。
- 实验采用多次运行取平均（基线3次，RL取最后4个检查点），报告标准差与相对变化，比较系统、客观。
- 针对RL训练稳定性、奖励设计的细粒度分析（如轨迹级 vs 轮次级奖励、重要性采样比上限γ的选择）也进行了补充实验，整体实验设计较充分。

### 6. 主要结论与发现
- Baselines普遍存在一致性-时机权衡矛盾：高动作一致性的模型往往过于保守，主动时机差；主动时机强则伴随不稳定和高误触发率。
- ProActor-Q4通过RL（特别是自定义/自适应RULER奖励）能显著提升主动时机（PT、RAR），同时保持有竞争力的动作一致性，且预测稳定性（差异度）远优于多数SOTA基线。
- 关键洞察：**RULER奖励中显式嵌入主动性评分细则**是提升时序质量的核心；**阶段感知的复合奖励**可有效平衡探索与优化目标。
- ART-F框架实现了在单节点多GPU上的高效量化模型RL训练，训练速度提升4~8倍，可为资源受限环境提供参考。

### 7. 优点
- **框架完整性**：从自动标注、系统评估到RL训练形成端到端闭环。
- **领域无关**：只需修改配置即可适配新领域，且在两类截然不同的数据集上验证。
- **评价体系系统**：同时涵盖动作一致性与时机质量，并通过PRI实现综合排序，有效揭示各种模型的行为特性。
- **RL奖励设计创新**：轮次级别、自定义RULER、阶段感知权重等设计具有可解释性与可控性。
- **基础设施贡献**：ART-F为单节点高效RL训练提供了实用参考，代码计划开源。
- 消融实验充分，结果分析细致，对方法论选择有合理解释。

### 8. 不足与局限
- **动作观察范围**：对齐验证仅依赖实际触发的动作，这仅是所有合理主动机会的子集，可能低估模型在更广泛机会上的表现。
- **语言与领域覆盖**：仅在英文数据集上评估，未涉及多语言或低资源语言，且对话长度限于≤50轮次，缺乏极长或多轮互动场景下的验证。
- **模型通用性**：RL训练只针对Qwen2.5-14B的量化LoRA变体，未扩展到其他模型家族（如Llama、Mistral）或不同参数量级。
- **ART-F扩展性**：目前仅支持单节点多GPU训练，尚未实现多节点分布式训练，大规模扩展仍需后续工作。
- **潜在偏差**：在金融等敏感领域，若训练数据存在服务差异，RL可能学习并复制这些不平等模式；实际部署需结合人工监督、公平性审计和用户同意机制。
