## 摘要
强化学习（RL）自然适用于智能体知识库问答（KBQA），在这种任务中，模型必须发出可执行的动作，观察知识库的反馈，并最终返回答案。然而，当前基于RL的KBQA系统主要优化来自最终答案的稀疏奖励，导致中间动作错误的监督较弱。这对带有逻辑形式标注的KBQA基准尤为限制：金标准逻辑形式可转换为可执行的动作序列，但现有方法主要将其用于热启动数据构建，而非进行策略内RL更新。我们提出GAPD，一种训练时金标准动作策略蒸馏框架，它为基于结果的RL添加了密集的词元级指导。为了将金标准动作与策略内的学生探索轨迹对齐，GAPD使用MID-ANCHOR MATCHING：它将学生探索和金标准执行过程中到达的中间实体视为状态锚点，并通过这些探索到的实体集将学生状态与金标准状态匹配。以对齐后的金标准动作为条件的当前策略充当停止梯度的教师，其词元分布被蒸馏回生成动作词元跨度上的普通学生策略。GAPD在WebQSP、GrailQA和GraphQ上一致超越了当前最先进水平。

## Abstract
Reinforcement learning (RL) is a natural fit for agentic knowledge base question answering (KBQA), where a model must issue executable actions, observe knowledge-base feedback, and eventually return an answer. However, current RL-based KBQA systems mainly optimize sparse rewards from the final answer, leaving intermediate action errors weakly supervised. This is especially limiting for logical-form annotated KBQA benchmarks: gold logical forms can be converted into executable action sequences, but existing pipelines use them mainly for warm-start data construction rather than for on-policy RL updates. We propose GAPD, a training-time Gold-Action Policy Distillation framework that adds dense token-level guidance to outcome-based RL. To align gold actions with on-policy student rollouts, GAPD uses MID-ANCHOR MATCHING: it treats the intermediate entities reached during student exploration and gold execution as state anchors, and matches student states to gold states through these explored entity sets. The current policy conditioned on this aligned gold action serves as a stop-gradient teacher, whose token distribution is distilled back to the ordinary student policy over generated action-token spans. GAPD consistently surpasses the current state of the art on WebQSP, GrailQA, and GraphQ.

- **整体含义**：论文旨在解决“如何将已有的金标准动作序列转化为强化学习训练中的密集、词元级监督信号”，以提升智能体中间动作的正确性，最终提高最终答案的准确率。

## 2. 论文提出的方法论
**核心框架：GAPD（Gold‑Action Policy Distillation）**，一种训练时的金标准动作策略蒸馏方法，其在传统的结果奖励（GRPO）基础上，增加了**基于实体锚点对齐的词元级指导**。

**关键模块与流程**：
1. **金标准动作序列获取**：将数据集提供的金标准逻辑形式解析为可执行的动作序列 \(g = (g_1, \dots, g_M)\)，并离线执行得到各步骤的中间状态及其到达的实体集合 \(E(u_m)\)。
2. **实体锚点匹配（ENTITY‑ANCHOR MATCHING）**：学生策略在线上探索时，每步生成一个动作，产生一个学生状态 \(s_t\) 和对应的实体集 \(E(s_t)\)。GAPD 计算学生状态与各金标准状态实体的 **Jaccard 相似度**：
   \[
   \text{sim}(s_t, u_m) = \frac{|E(s_t) \cap E(u_m)|}{|E(s_t) \cup E(u_m)|}
   \]
   选择相似度最高的金标准状态 \(m^*(t)\)。若匹配通过阈值（文中用 \(\tau_{state}=0.95\)），则认为学生状态已对齐到某个金标准执行点，下一步金标准动作 \(g_{m^*(t)+1}\) 就是该状态下的“有效动作提示”。
3. **金动作条件自教师（Gold‑Action‑Conditioned Self‑Teacher）**：
   - 在学生上下文 \(x_t\) 后**仅向教师侧**插入对齐的金标准动作文本，形成教师提示 \(\tilde{x}_t\)。
   - 用当前策略（冻结梯度）对该教师提示计算词元分布：
     \[
     p^{\text{teach}}_\theta(\cdot | \tilde{x}_t) = \text{stopgrad}(\pi_\theta(\cdot | \tilde{x}_t))
     \]
   - 同一策略在原始学生上下文下的分布为：
     \[
     p^{\text{stud}}_\theta(\cdot | x_t) = \pi_\theta(\cdot | x_t)
     \]
   - 这种设计不引入外部模型，而是让模型对照“如果已知正确动作我会怎么生成”来评价当前采样词元。
4. **词元级优势计算与融合**：
   - 对匹配步上学生生成的每个词元 \(y_{i,k}\)（在动作段上的词元），计算对数概率差：
     \[
     d_{i,k} = \log p^{\text{teach}}_\theta(y_{i,k} | \tilde{x}_{i,k}) - \log p^{\text{stud}}_\theta(y_{i,k} | x_{i,k})
     \]
   - 经裁剪和停止梯度后，得到词元级优势 \(A^{\text{gapd}}_{i,k}\)，仅在匹配到的词元位置上有效。
   - 与原始 GRPO 的轨迹级优势 \(A^{\text{grpo}}_i\) 相加：
     \[
     \hat{A}_{i,k} = A^{\text{grpo}}_i + \lambda_{\text{gapd}} A^{\text{gapd}}_{i,k}
     \]
   - 再用 PPO 式截断损失进行策略更新。
5. **安全过滤**：
   - 跳过答案生成步（已有结果奖励监督）。
   - 若学生动作执行后到达的实体集与对齐的金标准动作执行后的实体集相同，则丢弃该步的蒸馏信号，避免将等效路径强行拉到标注的表面形式，保留探索多样性。
6. **训练算法**：整体流程为：生成学生轨迹 → 执行学生与金标准动作，收集实体状态 → 用实体锚点匹配确定对齐步 → 构建教师提示并计算词元级优势 → 与 GRPO 优势融合后更新策略。

## 6. 论文的主要结论与发现
- GAPD 在所有三个基准及 GrailQA 的所有子集上均取得最优 EM 和 F1，相对于强基线 KBQA‑R1 有 2.4%–5.6% 的相对提升，尤其在组合和零样本场景下提升更显著。

## 7. 优点
- **创新性利用标注资源**：将 KBQA 中闲置的金标准逻辑形式，通过可执行的中间实体状态对齐，转化为 RL 的密集教师信号，填补了现有工作的空白。
- **巧妙的匹配与教师构造**：
  - 实体锚点匹配利用知识库执行反馈（实体集）作为状态表征，简单有效，无需额外模型。
  - 自教师机制避免引入外部模型，保持蒸馏信号与当前策略一致，符合 on‑policy 蒸馏思想。
- **非侵入式集成**：该方法作为 GRPO 的词元级优势增强插件，不改动原有奖励结构和 PPO 更新形式，易于集成到现有 RL‑KBQA 流程中。
- **实验扎实**：覆盖多个主流基准，对比丰富，消融及分析透彻，且披露了训练开销与推理效率权衡。
- **实用价值**：提升后策略的交互次数更少，对部署友好。

## 8. 不足与局限
- **训练开销**：比纯结果 RL 慢约 50%，需要额外的教师前向计算，虽可控但仍是成本。
- **依赖金标准逻辑形式**：方法假定 KBQA 数据集已完整标注了可执行的金动作序列。若标注不完整或缺失（如只有弱监督答案），则无法直接使用，限制了其扩展到无标注场景的可能。
- **自教师的局限性**：自教师知识完全来源于当前策略，可能无法纠正模型本身不具备的知识错误，只是迫使策略在给定金动作条件下自我修正偏好。
- **模型规模单一**：实验均基于 Llama‑3.1‑8B，未验证在不同规模、不同架构模型上的泛化性。
- **环境假设**：要求金标准动作序列在相同的知识库环境中完全可执行，并且实体锚点基于中间实体集的精确匹配（Jaccard 阈值高）。若金标准动作因 KB 更新或其他原因不可执行，匹配可能失败。
- **潜在评测偏差**：与闭源 LLM 的对比仅在 500 示例子集上进行，且这些 LLM 未做任何微调，对比可能低估了闭源模型的潜力（但论文已如实说明）。
- **动作空间覆盖**：当前动作集是固定的一组 KB 操作，扩展到更丰富或开放式的工具调用场景时尚需验证。
