## 摘要
结构化技能提示改进了长期智能体强化学习（RL）中的探索。技能增强的RL方法在推理时保留外部技能，而技能内化的RL方法在训练期间收回它们以实现自主性能。然而，现有的内化方法仅使用技能帮助性对比进行课程控制，策略更新保持不变，无法区分技能依赖和自主成功。我们提出 SkillC，一个基于对比技能信用分配（CSCA）的框架，该框架将此对比转化为用于内化的直接学习信号。SkillC 在同一策略更新内，针对来自活跃技能类型的任务，采样成对的技能注入和无技能的回滚，并通过双流优势估计器将其任务级对比注入优化，该估计器保持全局排名，同时对无技能成功应用单侧校正。平滑的验证级信号进一步驱动了一个自适应课程，涵盖归因强度、回滚分配和单调活跃集剪枝。在 ALFWorld 和 WebShop 上的实验表明，在没有运行时技能访问的情况下，SkillC 分别比先前最强的技能内化RL基线高出5.5%和4.4%，同时保持与技能增强RL方法的竞争力。

## Abstract
Structured skill prompts improve exploration in long-horizon agentic reinforcement learning (RL). Skill-augmented RL methods retain external skills at inference, while skill-internalization RL methods withdraw them during training to enable autonomous performance. However, existing internalization approaches only use skill-helpfulness contrast for curriculum control, leaving the policy update unchanged and unable to distinguish skill-dependent from autonomous success. We propose SkillC, a framework based on Contrastive Skill Credit Assignment (CSCA) that converts this contrast into a direct learning signal for internalization. \textsc{SkillC} samples paired skill-injected and skill-free rollouts for tasks from active skill types within the same policy update, and injects their task-level contrast into optimization via a dual-stream advantage estimator that preserves global ranking while applying a one-sided correction toward skill-free success. A smoothed validation-level signal further drives an adaptive curriculum over attribution strength, rollout allocation, and monotonic active-set pruning. Experiments on ALFWorld and WebShop show that, without runtime skill access, SkillC surpasses the strongest prior skill-internalization RL baseline by 5.5\% and 4.4\%, respectively, while remaining competitive with skill-augmented RL methods.

- **核心问题**：现有技能内化类强化学习方法（如 SKILL0）仅将“技能是否有用”的对比信息用于控制技能的逐步撤出（课程控制），但策略更新的目标函数并未利用这一对比。在每轮参数更新中，所有轨迹（rollouts）共享同一种提示条件（要么都有技能，要么都没有），导致优化过程无法区分某次成功究竟是因为依赖技能，还是已经实现了自主。这种 **“内化盲区”（internalization blindness）** 造成训练时的优化目标与推理时所需的自主能力不匹配：技能撤出后，任务成功率大幅下降。
- **整体含义**：论文提出，技能内化的关键是**在策略优化阶段就注入“技能依赖程度”的对比信号**，让模型在训练时就主动学习“不使用技能也能成功”，而不仅仅是通过课程撤出被动转移。SkillC 框架正是通过对比信用分配（CSCA）将这种对比转化为直接的学习信号，实现从“技能增强”到“自主能力”的平滑过渡。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程
- **核心思想**：在同一策略更新轮次中，针对同一任务**同时采样两种 Rollout**：注入技能的和不注入技能的，并利用两者的表现差距来重新分配信用，显式地鼓励无技能条件下的成功。
- **关键技术细节**：
  - **配对对比回滚（Paired Contrastive Rollouts）**：对每个活跃任务类型，按一定比例 ρ 分别生成技能注入（z=1）和无技能（z=0）的回滚，计算批次级任务对比差距 ˆ∆(x) = EMA(μ⁺(x) − μ⁻(x))，并做裁剪。该信号仅用于信用分配和课程控制，不作为奖励改造。
  - **双流优势估计（Dual‑Stream Advantage Estimation）**：
    - **流1**：保留全局排序，对所有轨迹进行组归一化 `Φ(Rᵢ) = (Rᵢ−μ_ep)/(σ_ep+ε)`，维持整体质量信号。
    - **流2**：按条件进行独立的归一化，并施加**单侧校正**：从技能注入轨迹的优势中减去 C(x)，从无技能轨迹的优势中加上 C(x)，其中 `C(x) = λ_eff(x)·[ˆ∆(x)]₊`。当 ˆ∆(x) > 0 时（技能仍然有用），这会提升无技能成功的优势，惩罚技能依赖的成功，将学习目标直接定义为“无技能时成功”。
  - **内化感知型课程**：每隔 d 步在验证集上计算平滑的对比差距 `¯∆_val(k)`，经 sigmoid 变换为门控值 `gate(k)`，同时控制：
    1. **归因强度** λ_dyn = λ·gate；
    2. **回滚分配比例** ρ_with 随 gate 下降而降低（但保持最低探索预算）；
    3. **活跃技能集单调剪枝**：当 `¯∆_val` 低于退休阈值时移除技能类别，且永不重新激活。
- **训练目标**：仍然遵循 GRPO 框架（PPO‑style 剪切 + KL 惩罚），但将原始组归一化优势替换为上述 CSCA 优势估计器。

### 8. 不足与局限
- **对技能库结构的依赖**：方法假设训练前已定义好固定的、与任务类型对齐的技能列表。在技能质量动态变化或技能与任务映射不确定的开放场景中，课程门控可能导致不稳定的退休决策。
- **验证信号的瓶颈**：内化进度估计依赖足够的验证覆盖率。当存在极端类别不均衡或罕见任务时，平滑后的验证对比差距可能噪声较大，提前触发技能退休。

- **额外的早期计算开销**：配对回滚带来约 26% 的早期算力成本，对于超大规模训练仍有一定负担。
