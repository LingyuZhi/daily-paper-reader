## 摘要
通过监督微调（SFT）和强化学习（RL）进行的后训练对于增强多模态大语言模型（MLLMs）的推理能力至关重要，然而现有范式常因静态数据的局限性而遭遇性能瓶颈。当前方法虽借助自反思或自演化突破这些局限，但仍受低质量合成数据导致的认知漂移和幻觉推理路径困扰。为应对这些挑战，我们提出锚点演化（AnE），一种融合真实锚定数据策展与模型演化的新范式，在推理前沿实现忠实且稳定的性能提升。具体而言，我们提出真实锚点扩展，通过轨迹展开精准定位模型失败前沿，并利用真实数据库检索高保真锚点进行忠实的数据策展。随后，我们引入支架剥离机制以内化推理能力。该机制首先通过支架增强监督锚定推理路径，以缓解直接对原始数据应用SFT的学习复杂性和分布漂移，然后利用RL剥离支架模板，从而有效将推理路径转化为模型内在能力。在多种多模态推理基准上的实验结果表明，我们的方法显著推进了模型性能前沿，在八个多模态基准上将基座模型提升10.3%，并取得了最先进结果。代码将公开提供。

## Abstract
Post-training via Supervised Fine-Tuning (SFT) and Reinforcement Learning (RL) is crucial for enhancing reasoning in Multimodal Large Language Models (MLLMs), yet existing paradigms often reach a performance bottleneck due to the limitations of static data. While current methods leverage self-reflection or self-evolution to push these boundaries, they still suffer from cognitive drift and hallucinated reasoning paths caused by low-quality synthetic data. To address these challenges, we propose Anchor Evolution (AnE), a new paradigm that integrates truth-anchored data curation and model evolution, achieving faithful and steady performance gains at the reasoning frontier. Specifically, we propose Truth Anchor Expansion, which pinpoints the model failing frontier via trajectory rollouts and leverages ground-truth databases to retrieve high-fidelity anchors for faithful data curation. Subsequently, we introduce the Scaffold-Stripping Mechanism to internalize reasoning capabilities. This mechanism first anchors reasoning paths via scaffold-augmented supervision to mitigate the learning complexity and distribution drift of direct SFT on raw data, then leverages RL to strip the scaffold template, thereby effectively transitioning the reasoning paths into intrinsic model capabilities. Experimental results on multimodal reasoning benchmarks show that our method substantially advances the model performance frontier, improving the base model by 10.3\% across eight multimodal benchmarks and achieving state-of-the-art results. The code will be made publicly available.

### 2. 论文提出的方法论
- **核心思想**：AnE通过“真实锚点扩展”与“支架剥离机制”的结合，形成一种可迭代的闭环训练范式。它精准定位模型无法解决的问题，从真实数据库中检索高质量样本来扩展数据，并通过支架引导和剥离，将外部知识内化为模型的内在能力。
- **关键技术细节**：
    - **1. 失败前沿发现 (Failing-Frontier Discovery)**：
        1.  **初步强化学习 (Preliminary RL)**：首先用一个种子RL数据集对模型进行RL训练，激活其潜在推理能力。
        2.  **数据路由 (Data Routing)**：对种子SFT数据集中的每个样本`x`进行`K`次随机展开，根据正确回答次数`c(x)`，将样本分为三类：
            - `c=K`：冗余样本，直接剪枝。
            - `1 ≤ c < K`：不稳定样本，保留用于SFT以维持稳定性。
            - **`c=0`：失败前沿样本**，这是模型的完全失败区域，是进化的关键目标。同时，RL训练中持续失败的样本也被归入此类。
        3.  **教师模型诊断 (Diagnostic Analysis)**：使用一个教师模型对前沿样本的失败轨迹进行分析，输出两类元数据：
            - **搜索关键词 `k_x`**：用于从外部数据库中检索相关真实样本。
            - **可操作提示 `h_x`**：用于降低问题难度、引导模型找到正确路径的脚手架。
    - **2. 真实锚点扩展 (Truth Anchor Expansion)**：
        1.  **检索真实锚点**：利用失败样本的搜索关键词`k_x`作为查询，从大规模真实数据库 `D_pool`中，通过预训练嵌入模型检索与其语义和逻辑上相似的Top-N个样本，作为高保真度的“真实锚点”`a_x`。
        2.  **数据合并与再路由**：将检索到的锚点数据去重后并入SFT数据集，并再次进行多轮展开和路由分析，保留其中的不稳定样本。
    - **3. 支架剥离机制 (Scaffold-Stripping Mechanism)**：
        1.  **支架增强SFT (Scaffold-Augmented SFT)**：针对失败前沿样本，将可操作提示`h_x`附加在原问题上，引导模型生成正确的推理轨迹。成功后，**将提示从输入问题端“搬迁”到输出回答的起始位置**，构成“支架增强”的SFT训练数据。这使得模型学习在其能力边界附近，通过提示来解决问题。
        2.  **通过RL剥离进行内化 (Internalization via RL Stripping)**：将失败前沿样本的**原始问题（不含提示）**加入RL数据集。模型必须在没有脚手架的情况下，独立得出正确答案。RL训练强制模型将之前依赖外部提示的推理路径，内化为自身的固有能力，从而真正扩展能力边界。
    - **迭代演化循环 (Iterative Evolution Cycle)**：将上述SFT和RL阶段作为一个完整周期，循环执行，每轮都以经过上一轮SFT和RL后的模型为起点，并重新发现新的失败前沿。
- **算法流程（文字说明）**：
    1.  **预热**：对基座模型进行一轮初步RL。
    2.  **循环开始**：
        a. **发现前沿**：在当前SFT数据集上进行`K`次展开，找出`c=0`的失败前沿样本。
        b. **诊断**：教师模型分析失败原因，生成`搜索关键词`和`可操作提示`。
        c. **锚点扩展**：用`搜索关键词`从真实数据库检索`真实锚点`数据，扩充SFT数据集。
        d. **构建支架数据**：用`可操作提示`引导模型生成正确解答，将提示搬到答案开头，形成“支架增强”SFT数据。
        e. **SFT训练**：模型使用`不稳定样本`和`支架增强样本`进行监督微调。
        f. **RL内化**：将不带提示的`失败前沿样本`加入RL数据集，进行强化学习训练，剥离支架依赖。
    3.  **循环结束**：进入下一轮迭代，从步骤a开始。
