## 摘要
记忆增强的LLM智能体通过将交互轨迹递归总结为紧凑记忆来解决复杂的长期任务。然而，现有方法通常使用基于结果的强化学习来训练这些记忆策略，未能定位中间记忆质量退化的位置。随着交互的进行，模糊的递归总结逐渐丢弃任务相关信息，引入语义噪声。这加剧了信念偏差，模糊了智能体对潜在任务状态的估计，最终使长期推理偏离轨道。因此，我们认为记忆优化不应仅关注轨迹层面的成功，还应关注中间总结所诱导的信念清晰度。为此，我们引入了信念熵，一种自监督代理，用于探测模型在给定当前记忆的情况下对潜在任务状态的不确定性。基于这种代理，我们提出了元认知记忆策略优化（MMPO）。MMPO 不仅依赖稀疏的基于结果的信号，还通过显式惩罚导致高认知不确定性的总结，提供细粒度、针对记忆的监督。实验表明，MMPO 在多种长期任务上始终优于现有方法，即使扩展到 175 万令牌上下文，仍保持 97.1% 的性能。

## Abstract
Memory-augmented LLM agents tackle complex long-horizon tasks by recursively summarizing interaction trajectories into compact memory. However, existing approaches typically train these memory policies using outcome-based reinforcement learning, failing to localize where intermediate memory quality degrades. As interactions unfold, ambiguous recursive summaries progressively discard task-relevant information and introduce semantic noise. This exacerbates belief deviation, obscuring the agent's estimate of the latent task state and ultimately derailing long-horizon reasoning. We therefore argue that memory optimization should focus not merely on trajectory-level success, but on the clarity of the belief induced by intermediate summaries. To this end, we introduce Belief Entropy, a self-supervised proxy that probes how uncertain the model remains about the latent task state given its current memory. Based on this proxy, we propose Metacognitive Memory Policy Optimization (MMPO). Instead of relying only on sparse outcome-based signals, MMPO provides fine-grained, memory-specific supervision via explicitly penalizing summaries that induce high epistemic uncertainty. Experiments show that MMPO consistently outperforms existing methods on diverse long-horizon tasks, maintaining 97.1% performance even when scaled to 1.75M-token contexts.

- **问题背景**：LLM智能体在解决长期任务（long-horizon tasks）时，受限于有限的上下文窗口和“迷失在中间”（lost-in-the-middle）现象，常采用**记忆增强型智能体（memory-augmented agents）** 范式，通过递归总结交互轨迹，将长历史压缩为紧凑文本记忆。
- **核心痛点**：递归总结过程会逐步**积累语义噪声**，丢弃关键信息，引发**信念偏差（belief deviation）**——即智能体对潜在任务状态的内部估计逐渐偏离真实状态，最终导致长期推理崩溃。
- **现有方法局限**：现有研究主要使用**基于结果的强化学习（RLVR）** 训练记忆策略，但奖励信号过于稀疏，存在严重的**信用分配（credit assignment）问题**，无法精准定位中间记忆质量下降的位置，缺乏对中间总结的细粒度监督。
- **研究动机**：论文主张，记忆优化不应该只盯着最终任务结果，而应该重点关注**中间总结所诱导的信念清晰度**，为记忆管理提供过程性、精准的优化信号。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程

- **核心思想**：将记忆优化建模为一个**信念保持（belief preservation）** 问题。引入一个基于**元认知探针（metacognitive probe）** 的自监督信号，来定量评估中间记忆的模糊程度，并以此作为密集奖励，引导策略生成更清晰的记忆总结。

- **关键技术细节——信念熵（Belief Entropy）**：
    - **理论框架**：将长期推理视为**部分可观测马尔可夫决策过程（POMDP）**。在记忆增强智能体中，真实状态 \( s_t \) 不可观测，智能体的**信念 \( b_t \)** 完全由其生成的文本记忆 \( m_t \) 所诱导（\( b_t = P(s_t | m_t) \)）。优化目标就是最小化潜在状态的条件熵 \( H(s_t | m_t) \)。
    - **实践代理**：由于 \( s_t \) 不可直接测量，提出**信念熵 \( H_{\text{BE}}(m_t) \)**，用于衡量智能体对自身记忆的不确定性。具体实现为：智能体根据当前记忆 \( m_t \)，回答一个**锚定问题（anchor question）**，然后计算模型响应 \( y \) 的条件熵：\( H_{\text{BE}}(m_t) = H(y | m_t, q) \)。
    - **锚定问题设计**：采用一种双重探针问题，同时探测“当前任务进度”和“仍缺失的信息”。这是一种将不可观测的信念不确定性转化为可观测响应不确定性的方法。计算上，通过生成贪婪响应并计算其令牌级平均预测熵来估计。

- **算法流程——元认知记忆策略优化（MMPO）**：
    1. **轨迹采样**：记忆策略 \( \pi_\theta \) 为每个任务采样 \( N \) 条完整交互轨迹。
    2. **子轨迹奖励计算**：对于每条轨迹的第 \( k \) 步，计算一个既包含局部记忆质量又包含全局结果的**密集奖励**：
       \[
       R^{(i)}_k = \alpha \cdot \sigma(-H_{\text{BE}}(m^{(i)}_k)) + r^{(i)}_{\text{final}}
       \]
       其中，\( \sigma \) 是sigmoid函数，用于将信念熵归一化到 \( (0,1) \) 区间；\( r_{\text{final}} \) 是最终结果奖励（如F1分数）；\( \alpha \) 控制信念奖励的权重。这为每个中间步骤都提供了训练信号。
    3. **回合级优势估计**：采用**组相对优势（Group Relative Advantage）** 方法，在组内对同一深度的子轨迹奖励进行标准化，得到子轨迹优势。然后，将智能体在时刻 \( t \) 生成的记忆 \( m_t \) 所影响的所有未来子轨迹优势进行聚合，计算**回合级优势 \( A^{(i)}_t \)**：
       \[
       A^{(i)}_t = \frac{1}{T-t+1} \sum_{k=t}^T \hat{A}^{(i)}_k
       \]
    4. **策略优化**：使用经典的**PPO截断目标**，并加入KL散度惩罚，来优化记忆策略 \( \pi_\theta \)。
