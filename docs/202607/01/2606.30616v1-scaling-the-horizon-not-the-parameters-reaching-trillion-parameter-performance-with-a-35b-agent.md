## 摘要
我们提出了Agents-A1，一个35B参数的混合专家（MoE）智能体模型，通过扩展智能体视野达到了万亿参数级别的性能。我们从两个角度研究智能体视野扩展：扩展长视野轨迹和扩展异构智能体能力。为支持这一目标，我们构建了一个长视野知识-动作基础设施，连接外部知识、动作、观察和验证器结果，生成平均长度为45K tokens的智能体轨迹。基于此，我们采用三阶段策略训练Agents-A1。首先，进行全领域监督微调，使基础模型与广泛的智能体行为对齐。其次，训练领域级教师模型，捕获每个领域的专业知识。第三，我们提出了一种多教师领域路由的在线蒸馏方法，结合显著词汇对齐，以提高不同领域间的知识迁移效率，将六个异构领域统一到一个可部署的学生模型中。Agents-A1在长视野智能体基准测试中展现出强大且广泛的表现。与万亿参数模型如Kimi-K2.6和DeepSeek-V4-pro相比，Agents-A1在SEAL-0（56.4）、IFBench（80.6）、HiPhO（46.4）、FrontierScience-Olympiad（79.0）和MolBench-Bind（56.8）上取得领先结果，并在SciCode（44.3）、HLE（47.6）和BrowseComp（75.5）上保持高度竞争力。我们希望这项工作为社区提供一条使用35B智能体在实际中扩展视野的路径，使其在长视野任务上达到或匹配万亿参数模型的性能。

## Abstract
We introduce Agents-A1, a 35B Mixture-of-Experts Agentic Model that reaches trillion-parameter-level performance by scaling the agent horizon. We investigate agent-horizon scaling from two perspectives: scaling long-horizon trajectories and scaling heterogeneous agent abilities. To support this goal, we build a long-horizon knowledge-action infrastructure that connects external knowledge, actions, observations, and verifier outcomes, producing agentic trajectories with an average length of 45K tokens. Based on this, we train Agents-A1 with a three-stage recipe. First, we perform full-domain supervised fine-tuning to align the base model with broad agentic behaviors. Second, we train domain-level teacher models to capture specialized expertise in each domain. Third, we propose a multi-teacher domain-routed on-policy distillation with salient vocabulary alignment to improve knowledge transfer efficiency across different domains, unifying six heterogeneous domains into one deployable student model. Agents-A1 achieves strong and broad performance for long-horizon agent benchmarks. Compared with 1T-parameter model such as Kimi-K2.6 and DeepSeek-V4-pro, Agents-A1 achieves leading results on SEAL-0 (56.4), IFBench (80.6), HiPhO (46.4), FrontierScience-Olympiad (79.0), and MolBench-Bind (56.8), and remains highly competitive on SciCode (44.3), HLE (47.6) and BrowseComp (75.5). We hope this work provides the community with a practical path for scaling the horizon using a 35B agent that can reach or match the performance of 1T models on long-horizon tasks.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：当前大语言模型朝向智能体方向发展，但主流路线依赖**扩展模型参数**（如万亿参数模型）来获取广泛推理、工具使用和领域知识能力。这种路线成本高昂、复现困难，且对中小型研究团队不友好。
- **核心动机**：探索**通过扩展智能体视野（horizon）而非参数**来提升性能。具体而言，让智能体在执行长周期任务时能够显式地进行知识获取、动作执行、观察解释和验证，从而利用过程监督信号取代单纯增大模型规模。
- **挑战**：
  - **基础设施瓶颈**：缺乏统一的知识-动作交互基础设施来生成和利用长视野轨迹（平均45K tokens）。
  - **异质能力整合**：不同领域（如搜索、编程、科学研究、指令遵循）的推理和交互模式差异大，难以融合到一个统一模型。

## 2. 方法论：核心思想、关键技术细节
### 核心思想
- 构建**长视野知识-动作基础设施（Knowledge-Action Infrastructure）**，将外部知识、动作、观察和验证信号连接成图结构，生成高密度、可验证、可自扩展的监督数据。
- 设计**三阶段训练流程**：
  1. **全领域监督微调（SFT）**：在混合域长视野轨迹上微调基础模型（Qwen3.5-35B-A3B），获得广泛的智能体能力。
  2. **领域教师模型训练**：针对每个领域（搜索、机器学习工程、科学研究、指令遵循、工具调用等）训练专门的教师模型，通过SFT和强化学习（RL）进一步提升专长。
  3. **多教师在线蒸馏（On-Policy Distillation, OPD）**：使用领域路由+显著词汇对齐（Salient Vocabulary Alignment, SVA）将多个教师的能力蒸馏到一个统一的学生模型，减少域间冲突。

### 关键技术细节
- **知识-动作图（Knowledge-Action Graph, KAG）**：
  - 定义为一个四元组 \( G_d = (C_d, A_d, O_d, V_d) \)，分别表示域语料、动作空间、观察空间、验证器集合。
  - 每个动作记录为 \( (s_t, a_t, o_t, v_t) \)，保留中间步骤的证据和验证结果。
  - 通过**自博弈图搜索和扩展**（proposer–solver–verifier游戏）生成高质量子图。
- **多教师领域路由在线蒸馏（Multi-teacher Domain-routed OPD with SVA）**：
  - SVA：对于每个学生生成的位置，只在教师选定的top-k显著词汇上计算反向KL散度，避免全词汇对齐的噪声。
  - 域归一化目标：对每个域内样本的损失取平均，再对所有活跃域取平均，防止高频域主导更新。
- **各领域数据管道**：
  - 长视野搜索：基于Wiki构建实体-超链接图，生成路径搜索问答。
  - 机器学习工程：基于MLE-Dojo和Kaggle竞赛，使用树形代码探索、执行和验证。
  - 科学研究：从科学问题库扩展出15K高难度、高交互的问题，生成无工具和工具增强的推理轨迹。
  - 指令遵循：13K多约束指令+10K长上下文QA，注入局部上下文规则和干扰。
  - 工具调用：基于工具依赖图合成任务，在沙箱中生成多轨迹。

## 6. 主要结论与发现
- **Agents-A1（35B）在多个长视野基准上匹配甚至超越万亿参数模型**：

- **OPD后的模型不一定在每个域上都超过对应的专门教师**，但整体平衡性更好。
